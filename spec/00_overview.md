# Solder Fume Blower (USB-C)

Soldering fume extractor / blower with:
- Static pressure (sirocco BLDC fan)
- Speed control
- Auto-off (forget prevention)
- Replaceable filter cartridge
- USB-C powered (5V input -> 12V internal)
- Compact desktop form factor

## Fixed Components
- Fan: TOSHIBA CL-8028L-11 (12V 260mA, 3-wire)
- MCU: RP2040 (Dirtcheap chinese board)
